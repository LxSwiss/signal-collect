Let's get started!

Set up your development environment and run your first algorithm in less than 5 minutes!

Compiling the Project

Java 7

Ensure Java 7 is used for the compilation.

> java -version

Simple Build Tool (SBT)

Install SBT

Go to the project folder and start SBT on the command line.

> sbt

The output should end with the following line:

[info] Set current project to signal-collect (in build file:XYZ/signal-collect/)

Complile an executable .jar File

To generate a .jar file containing all the needed dependencies, use the "assembly" command on the SBT prompt.

Develop in Eclipse

To generate an Eclipse project, use the "eclipse" command on the SBT prompt.
